# CCUS-CT 前端界面

基于Vue.js构建的CCUS知识图谱对话系统前端界面。

## 功能特性

- 响应式聊天界面
- 知识图谱可视化展示
- 实时对话交互
- 图表数据展示

## 技术栈

- Vue 3
- Ant Design Vue
- D3.js (图谱可视化)
- ECharts (数据图表)
- Vite (构建工具)

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发环境启动
```bash
npm run dev
```

### 生产环境部署
```bash
npm run build
npm run server
```
